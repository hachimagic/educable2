# Educable Website
## Status
work in progress!

## Installation
### Frontend

Have NodeJS (18 or 20)

clone the repo down

- (npm) 
- `cd Frontend`
- `npm install`
- `npm run dev`


### Backend

- (npm) 
- `cd Backend`
- `npm install`
- `npm run dev`
or
- `npm run pdev` for multiprocessed server development
